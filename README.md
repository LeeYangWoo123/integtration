# 📟 스마트 함수 분석기 PRO (Smart Function Analyzer)

Streamlit과 SymPy를 활용한 지능형 함수 분석 및 적분 계산기입니다. 
뉴모피즘(Neumorphism) 디자인이 적용된 다크 모드 UI를 제공합니다.

### ✨ 주요 기능
- **수식 입력 및 파싱**: `2x`, `sin(x)` 등 자연스러운 수식 입력 지원
- **미분 및 적분**: 도함수 및 부정적분 결과 실시간 LaTeX 출력
- **정적분 계산기**: 특정 구간(a, b)의 면적 계산 및 시각화
- **그래프 시각화**: Matplotlib을 이용한 함수 그래프 및 적분 영역 강조
- **역함수 계산**: 입력된 함수의 역함수 자동 산출

### 🚀 실행 방법
1. 저장소 클론: `git clone https://github.com/사용자이름/저장소이름.git`
2. 패키지 설치: `pip install -r requirements.txt`
3. 앱 실행: `streamlit run app.py`
